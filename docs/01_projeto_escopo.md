# Escopo do Projeto

## 1. Problema de negócio

O hospital precisa de uma solução preditiva para apoiar profissionais de saúde na identificação de possíveis categorias de obesidade com base em dados clínicos, demográficos e comportamentais.

## 2. Problema de Machine Learning

Este projeto será tratado como um problema de classificação multiclasse, no qual o modelo recebe variáveis de entrada relacionadas ao indivíduo e retorna uma classe prevista de condição corporal/obesidade.

## 3. Variável alvo

A variável alvo é `NObeyesdad`, posteriormente renomeada para "nivel_obesidade", que representa a categoria relacionada ao nível de obesidade do indivíduo.

## 4. Métrica obrigatória

A métrica obrigatória do desafio é assertividade superior a 75%. Neste projeto, essa assertividade será interpretada como accuracy.

## 5. Critério mínimo de sucesso

O modelo final deve apresentar accuracy superior a 75% em dados de teste, mantendo desempenho aceitável nas métricas complementares.
