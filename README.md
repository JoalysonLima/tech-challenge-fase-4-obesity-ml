# Tech Challenge Fase 4 - Obesity Prediction

## 1. Visão Geral do Projeto

Este projeto visa desenvolver uma solução de Machine Learning para auxiliar profissionais de saúde na previsão do nível de obesidade de um paciente com base em dados demográficos, perfil corporal, de estilo de vida e comportamentais.

A solução inclui:
- um pipeline completo de Machine Learning;
- feature engineering;
- treinamento e avaliação do modelo;

- um aplicativo preditivo Streamlit;

- um painel analítico;

- um documento de entrega final com todos os links necessários.

## 2. Problema de Negócio
Um hospital precisa de uma ferramenta de apoio à decisão para ajudar os profissionais de saúde a identificar possíveis níveis de obesidade em pacientes. O sistema destina-se a apoiar a triagem e as discussões clínicas, não a substituir o diagnóstico médico.

## 3. Objetivo de Machine Learning
Construir um modelo de classificação multiclasse capaz de prever a categoria de obesidade de uma pessoa.

## 4. Dataset
Base utilizada: obesity.csv.

As variáveis ​​esperadas incluem características demográficas, perfis corporais e comportamentais, como idade, sexo, altura, peso, hábitos alimentares, consumo de água, atividade física, uso de tecnologia e meio de transporte.

## 5. Metodologia

O projeto segue estas etapas:

1. Diagnóstico de dados
2. Análise exploratória de dados
3. feature engineering
4. Pipeline de pré-processamento
5. Treinamento do modelo
6. Avaliação do modelo
7. Implantação do Streamlit
8. Desenvolvimento do painel analítico

## 6. Avaliação

A principal métrica exigida é a acurácia, com uma meta mínima de 75%.

Métricas adicionais:
- Balanced accuracy
- Recall
- F1-score
- Matriz de confusão
